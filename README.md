# Note-about-Java
### 資料型態 Data Types
1. 整數 : byte (-128-127), short (-32768-32768), **int** (-2147483648-2147483647), long (-9223372036854775808-9223372036854775808)  
2. 浮點數 : float (到小數點下7位), **double** (到小數點下15位)  
3. 布林值 : boolean (true/false)  
4. 字元 : char ('')  
5. 字串 : String("")
